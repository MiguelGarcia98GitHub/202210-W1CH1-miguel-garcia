HOW TO USE

1 ARG OPERATION:

Enter a value, then press CALCULATE

Expected result: Root Square of the value

2 ARGS OPERATION:

Enter a value, then press ADD A SPACE, then add another value, then press CALCULATE

Expected result: Sum, Subtraction, Multiplication, Division of the 2 values

3 OR MORE ARGS OPERATION:

Enter a value, then press ADD A SPACE, then add another value, then press ADD A SPACE, then add another value, and so on. When ready, press CALCULATE

Expected result: Sum of all values
